# Template: Vídeo para Kling 3.0

## Quando usar
- Criativos em vídeo para Meta Ads (Reels, Stories, Feed)
- Motion de produto para maior engajamento
- Narrativas curtas de 6 a 15 segundos

## Como o Kling funciona

- Gera clipes de até **10 segundos** por geração
- Funciona melhor com **1 ação principal por clipe**
- Descrever **câmera + sujeito + ação + ambiente** na mesma frase
- Evite cenas complexas com muitos elementos em movimento simultâneo

## Estrutura de roteiro para IA

Divida o vídeo em **cenas de 2–4 segundos**. Para cada cena, escreva o prompt separado.

```
CENA [N] — [2-4s]
Sujeito: [quem ou o quê está em cena]
Ação: [o que acontece, com verbo preciso]
Câmera: [tipo de movimento e ângulo]
Ambiente: [onde / contexto visual]
Prompt completo: [frase única otimizada para o Kling]
```

## Movimentos de câmera disponíveis no Kling

| Movimento | Descrição | Melhor para |
|---|---|---|
| Slow push in | Câmera avança lentamente para o sujeito | Produto hero, destaque emocional |
| Pull back reveal | Câmera recua e revela o ambiente | Abertura de cena, contexto |
| Pan left / right | Câmera desliza horizontalmente | Ambientes, linhas de produto |
| Drone shot descending | Vista aérea descendo | Feiras, ambientes grandes, scale |
| Orbit / arc shot | Câmera circula o sujeito | Produto 360°, destaque premium |
| Handheld subtle shake | Movimento orgânico leve | Lifestyle, autenticidade |
| Static locked off | Câmera parada, só o sujeito se move | Produto em fundo limpo, minimalismo |

## Exemplo de roteiro completo — Campanha Industrial (Samech FEIMEC)

```
CENA 1 — 3s | GANCHO VISUAL
Sujeito: Large black rubber O-ring industrial seal
Ação: Slowly rotating in mid-air
Câmera: Slow push in, macro level, orbit right
Ambiente: Dark studio background with rim lighting

Prompt Kling: Large black rubber industrial O-ring slowly rotating in mid-air, 
professional studio rim lighting from sides, dark navy background, 
camera slowly pushing in with subtle orbit right movement, 
macro close-up revealing surface texture, cinematic 4K, smooth motion

---

CENA 2 — 3s | CONTEXTO / DOR
Sujeito: Factory machinery, pipes with visible wear
Ação: Steam or fluid leaking from joint area, subtle
Câmera: Static locked off, medium close-up
Ambiente: Industrial plant, dim lighting

Prompt Kling: Industrial factory pipe joint with subtle steam leak, 
worn metal components, dim industrial lighting, 
static camera medium close-up, realistic, 
cinematic 4K, slightly desaturated, tense atmosphere

---

CENA 3 — 3s | SOLUÇÃO / PRODUTO
Sujeito: Technician's hands installing new seal on machinery
Ação: Precise installation movement, confident and quick
Câmera: Over-the-shoulder close-up, slight rack focus
Ambiente: Factory floor, better lit than cena 2

Prompt Kling: Technician's hands confidently installing black rubber seal 
on industrial machinery, close-up over-the-shoulder shot, 
slight rack focus from hands to installed seal, 
factory floor background softly lit, professional atmosphere, 
cinematic 4K, warm color grade

---

CENA 4 — 2s | CTA / MARCA
Sujeito: Clean branded frame with logo and booth number
Ação: Text and logo appearing with subtle fade
Câmera: Static, centered composition
Ambiente: Dark background with brand color

Prompt Kling: Clean dark navy background, brand logo and "FEIMEC M016" 
text appearing with elegant fade-in motion, 
minimal premium aesthetic, centered composition, static camera, 
subtle particle light effect, cinematic 4K
```

## Dicas para melhor resultado no Kling 3.0

1. **Uma ação por clipe** — cenas com 2+ ações simultâneas ficam inconsistentes
2. **Verbos de movimento precisos** — "rotating slowly", "pouring", "walking toward camera" > "doing stuff"
3. **Sempre inclua qualidade** — "cinematic 4K, smooth motion, professional" no final de cada prompt
4. **Iluminação descrita** — o Kling responde bem a "rim lighting", "softbox", "natural window light"
5. **Gere e refine** — gere 2-3 variações por cena e monte o melhor no editor

## Negative prompt padrão para Kling
```
jittery, flickering, distorted faces, extra limbs, artifacts, 
morphing background, inconsistent lighting, amateur, low quality
```
