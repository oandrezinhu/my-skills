# Template: Lifestyle / Ambiental

## Quando usar
- Criativos que mostram o produto em uso real
- Campanhas de identificação com o público
- Quando o contexto/ambiente conta parte da história

## Estrutura do prompt

```
[sujeito] [ação com o produto ou em contexto relevante],
[ambiente/cenário detalhado], [hora do dia / iluminação natural],
[composição: wide shot / medium shot / over-the-shoulder],
candid lifestyle photography, authentic moment, 
[mood: warm / energetic / calm / professional],
natural color grading, [paleta tonal],
shot on [câmera], [lente], f/[abertura], 8k, editorial quality
```

## Negative prompt padrão
```
stock photo look, forced smile, staged pose, ugly composition, 
cluttered background, harsh flash, oversaturated, fake, plastic,
extra limbs, distorted, watermark, text
```

## Dicas de uso no Freepik
- "Candid" e "authentic moment" são palavras-chave que evitam o look de banco de imagem
- Para industrial: sempre inclua equipamento/ambiente específico do setor — evita imagens genéricas
- Para eventos: adicione "motion blur in background, sharp foreground" para dar sensação de movimento
