# Template: Iluminação de Estúdio

## Quando usar
- Retratos de pessoas/personagens para campanhas
- Produto isolado com controle total da luz
- Quando precisa de consistência visual entre peças

## Estrutura do prompt

```
[sujeito], [expressão ou ação], professional studio photography,
[tipo de setup de luz: three-point lighting / Rembrandt lighting / butterfly lighting / rim lighting],
[fundo: seamless white / dark grey cyclorama / colored backdrop],
[lente e câmera], sharp focus, editorial quality,
[mood: confident / approachable / dramatic / clean], 
color graded in [tom de cor], 8k, commercial photography
```

## Tipos de iluminação e quando usar

| Setup | Visual | Melhor para |
|---|---|---|
| Three-point lighting | Limpo, comercial, sem sombras duras | Retratos corporativos, produto neutro |
| Rembrandt lighting | Sombra triangular no rosto, dramático | Campanhas de autoridade, B2B, luxo |
| Butterfly lighting | Sombra suave abaixo do nariz, glamour | Beleza, moda, lifestyle premium |
| Rim lighting | Borda iluminada, fundo escuro, destaque | Produto tech, industrial, impacto de scroll |
| Split lighting | Metade iluminada, metade sombra | Contraste, dualidade, impacto criativo |
| Softbox frontal | Uniforme, flat, seguro | E-commerce, produto simples |

## Exemplos

```
confident male engineer in his 40s, looking directly at camera, slight smile,
professional studio photography, Rembrandt lighting setup, 
dark grey seamless background, wearing industrial safety vest,
shot on Canon R5, 85mm f/1.4, sharp focus, commercial photography,
color graded in cool steel tones, authoritative mood, 8k
```

### Produto isolado premium
```
industrial rubber seal component, centered hero shot, floating on black background,

pure black seamless backdrop, subtle reflection on surface below,

```

### Retrato lifestyle (consumo)
```
young woman in her late 20s, laughing naturally, looking slightly off-camera,

seamless white background, wearing casual but stylish outfit,
shot on Sony A7R, 85mm f/1.8, shallow depth of field,

```

## Negative prompt padrão
```
harsh shadows, overexposed, flat lighting, amateur, grain, noise, 
motion blur, plastic skin, uncanny valley, distorted face, extra fingers,
watermark, text overlay
```

## Dicas de uso no Freepik
- Use o prefixo "professional studio photography" sempre — muda o comportamento do modelo
- Para retratos, adicione etnia/idade/gênero específicos se necessário para o público da campanha
- Gere sem roupa/acessório específico e adicione depois em edição ou nova geração com inpainting
