# Template: Imagem Vetorial / Ilustração

## Quando usar
- Folders, materiais impressos, banners institucionais
- Quando a identidade visual do cliente é baseada em ilustração
- Quando precisa de escalabilidade sem perda de qualidade

## Estrutura do prompt

```
[estilo de ilustração], [sujeito principal], [ação ou pose], [contexto/cenário], 
flat design vector illustration, clean lines, [paleta de cores], 
minimalist composition, geometric shapes, professional graphic design, 
high contrast, sharp edges, no texture, scalable art style,
[marca/segmento] aesthetic, editorial quality, 8k vector render
```

## Exemplos por segmento

### Industrial / B2B
```
flat design vector illustration, industrial engineer inspecting a large o-ring seal, 
factory background simplified, navy blue and steel grey palette with orange accent, 
clean lines, geometric shapes, professional graphic design, 
minimal texture, B2B corporate aesthetic, editorial quality, 8k vector render
```

### Varejo / Consumo
```
flat design vector illustration, happy customer holding product, 
modern urban background, vibrant coral and white palette with yellow accent, 
playful composition, rounded shapes, lifestyle brand aesthetic,
no texture, clean lines, high contrast, 8k vector render
```

### Saúde / Bem-estar
```
flat design vector illustration, person in relaxed wellness pose, 
minimal nature elements (leaf, circle), soft sage green and cream palette,
organic shapes, gentle composition, wellness brand aesthetic,
no texture, editorial quality, 8k vector render
```

## Negative prompt padrão
```
photorealistic, photo, 3d render, noisy, grain, blur, low quality, 
watermark, text, busy background, shadows, gradients
```

## Dicas de uso no Freepik
- Selecione estilo "Illustration" ou "Vector"
- Use aspect ratio 1:1 para feed, 9:16 para stories
- Gere 4 variações e escolha a base para refinar
